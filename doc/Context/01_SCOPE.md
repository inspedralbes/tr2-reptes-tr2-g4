# Project Scope

This document outlines the boundaries and key features of the platform.

## In Scope
- **Document Processing:** Support for PDF, Word, and XML uploads.
- **AI Integration:** Extraction of key educational adaptations using local LLMs (Ollama).
- **Teacher Dashboard:** A web interface for viewing summarized student profiles.
- **Real-time Communication:** Notification system using WebSockets (Socket.io).
- **Secure Storage:** Intermediary repository for encrypted/anonimized data.
- **Role-Based Access Control (RBAC):** Specific views for teachers, tutors, and admins.

## Out of Scope (Current Phase)
- Integration with official government education databases (SAGA/Esfer@).
- Mobile application (currently web-only).
- Direct editing of the original school documents within the platform.
